<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the Repository</p>

```
git clone https://github.com/nilaachandra/27-startups
```

<p>2. Install the dependencies</p>

```
npm install
```

<p>3. Run the Project</p>

```
npm run dev
```
